## メインデスク
U-POWER ゲーミングデスク
* 幅140cm
* 奥行60cm

横幅が広いのはかなり快適、奥行きは狭いが、モニターを浮かせることで、奥にキーボードを置けて今のところ問題ない。

デスク自体に配線用の穴が開いているので、配線を隠したい人にとってはよいと思う。

デスクが中央で半分に割れているので、デスクマットは必須(付属品あり)

## メインPC

* Windows11 Pro
* Intel Core i7 12700K
* DDR4 32GB Mem 3200 GHz
* 2TB SSD
* (RTX3060Ti ほしい)

CPUもメモリもこのクオリティならほぼ満足。今のところGPUは入れてないけど、動画や3Dデータを扱ってもそこそこ快適。機械学習をぶん回すときに購入すると思う。

## サブPC
